# SecureFile-Transfer-App
Designed a scalable backend architecture for secure and efficient file transfer similar to cloud-based storage systems.

Tech Stack: MongoDB (GridFS), Express.js, React.js, Node.js, JWT, REST APIs

Project Description:
Developed a secure file transfer web application using the MERN stack that enables users to upload, store, and share files safely over the internet. Implemented MongoDB GridFS for efficient storage and management of large files. The system supports secure file access using unique generated keys and controlled download permissions.

Key Features:

   Built a full-stack application using React frontend and Node.js + Express backend.
 
   Implemented MongoDB GridFS for storing large files efficiently.

   Secure file sharing using auto-generated access keys.

   RESTful APIs for file upload, download, and management.

   File metadata management including upload time and ownership.

   Automatic file deletion mechanism using scheduled jobs (Cron).

   Authentication and authorization using JWT.

   Responsive and user-friendly UI.

Impact / Learning Outcomes:

  Improved understanding of backend architecture and secure file handling.

  Hands-on experience with large file storage systems.

  Implemented real-world API security and database integration.
